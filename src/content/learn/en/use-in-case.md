## Using documents in a case

Once a document is generated, case workers need to access it. Valtimo provides built-in UI components for working with Epistola documents directly within a case.

### The documents tab

Every case can show a documents tab listing all generated documents. Each entry shows:

- Document name and type
- Generation timestamp
- Status (completed, failed)
- Template and variant used

### Preview

Click any document to see a PDF preview inline — no download required. This lets case workers verify the document before sending it to a citizen or archiving it.

### Download

Download the PDF to save it locally, attach it to an email, or upload it to an external system like OpenZaak for archival.

### In the process

Beyond the documents tab, you can add preview and download buttons directly into process forms. This means case workers see the relevant document right where they're working — no need to navigate to a separate tab.
