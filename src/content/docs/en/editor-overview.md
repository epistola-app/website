## Editor Overview

The Epistola Suite includes a rich document template editor built with Lit and ProseMirror web components.

### Split view

The editor provides a split-view layout:

- **Left pane** — Template structure and content editing
- **Right pane** — Live preview with sample data

Changes in the editor are immediately reflected in the preview, giving authors instant feedback on their document design.

### WYSIWYG editing

The editor is fully WYSIWYG (What You See Is What You Get). Authors work with a visual representation of the document rather than raw markup. Formatting, layout, and data bindings are all applied visually.

### Autosave

Draft versions are automatically saved as authors work. There's no need to manually save — the editor persists changes continuously so work is never lost.

### Drag-and-drop

Content blocks can be rearranged using drag-and-drop. Authors can:

- Reorder blocks within a section
- Move blocks between sections
- Insert new blocks from the block palette

This makes layout adjustments intuitive and fast.
