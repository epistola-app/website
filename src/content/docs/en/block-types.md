## Block Types

The Epistola editor provides a variety of content blocks for building document templates.

### Text

Rich text blocks with full formatting support — headings, paragraphs, bold, italic, lists, and inline expression chips for dynamic data.

### Table

Static tables with configurable rows and columns. Cell content supports rich text and expressions.

### Data table

Dynamic tables that repeat rows based on array data. Define the columns and bind each to a data path — Epistola generates one row per array item.

### Image

Insert static images or dynamically reference images from your data payload. Supports configurable sizing, alignment, and alt text.

### QR code

Generate QR codes from data values. Useful for linking physical documents to digital resources or tracking IDs.

### Containers and columns

Layout blocks that group other blocks together. Columns allow side-by-side arrangements within the document.

### Conditional

Show or hide blocks based on expression evaluation. Use conditions to create templates that adapt to different data scenarios.

### Loop

Repeat a set of blocks for each item in an array. Loops work with any data path that resolves to an array.

### Page break

Force a page break at a specific point in the document.

### Headers and footers

Define page headers and footers with support for page numbers, dates, and other dynamic content. Headers and footers can differ between the first page and subsequent pages.
