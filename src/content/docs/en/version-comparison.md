## Version Comparison

Version comparison lets authors visually compare two versions of a variant side by side, making it easy to review changes before publishing or deploying.

### How it works

The comparison view renders two versions of the same variant in parallel and displays both PDFs in a fullscreen dialog. Authors can see exactly what changed between versions.

### Comparison dialog

The fullscreen dialog provides:

- **Version selectors** — Choose which two versions to compare (e.g., v3 vs v4)
- **Data example selector** — Pick which named data example to use for rendering both previews
- **Side-by-side PDFs** — Both versions are rendered in parallel and displayed in iframes

### Access points

Version comparison can be accessed from:

- **Deployment matrix** — The Compare button on any cell lets you compare the deployed version against another version
- **Version history** — Compare any two versions from the version history dialog
